Enterprise Sales & Revenue Intelligence

Description: Developed a robust sales analytics platform handling $109M+ in revenue and 121K+ orders. The solution is built on a sophisticated Star Schema data model to integrate sales, product, and territory data. Key features include an AI-powered Q&A visual for natural language queries and dynamic timelines to track order fulfillment against shipping deadlines.

Tech Stack: Power BI, DAX, Data Modeling (Star Schema), Power Query.
